# Hotel Management System

## How to Run

1. Open the terminal and navigate to the project folder:
   ```bash
   cd hotel-management-system
   ```
2. Install Flask:
   ```bash
   pip install flask
   ```
3. Start the application:
   ```bash
   python app.py
   ```
